## Commands:

#### **Install Dependencies**

```
npm install or yarn
```

#### **Serve For Development**

```
npm start or yarn start
```

## Features

- Account Exporting (exports.txt)
- Auto Email Verification (IMAP)
- Uses Playwright to generate accounts
- Custom Config. (config.json)
- Catchall Based
